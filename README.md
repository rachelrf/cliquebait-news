# CliqueBait

Topic-specific news aggregator

## Usage

Development:
1. Install dependencies.
2. Run npm start to run server.
3. In a new tab, run mongodb.
4. Server will be listening on localhost:3000.

Production:
Visit [Cliquebait](https://cliquebait.herokuapp.com/)!
